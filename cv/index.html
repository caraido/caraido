Projects

  • 2019.03 - 2019.12
    Stages of Parkinson's Disease Detection with mobile phone video recordings
    Parkinson's Disease diagnosis requires clinical assessments that may not be frequent enough to track its progression. This project aims to develop a machine learning pipeline for the detection and staging of Parkinson's disease using video recordings captured on mobile phones.
    • Machine learning pipeline development
    • Pose estimation with OpenPose
    • Kinematic feature engineering
  • 2020.02 - 2020.06
    Subsecond Animal Behavior Embeddings classification with Variational Autoencoders
    Ethology studies heavily rely on human labeling of the data. This project improves the algorithm of an unsupervised learning model that embeds subsecond animal behavior (TREBA) that allows interpretable clustering of mice behaviors.
    • Variational Autoencoder model improvement
    • Triplet loss
    • Keypoint detection with DeepLabCut
    • Kinematic embedding
  • 2024.06 - present
    The emergent effects of lexical and phonemic bias in automated speech recognition models
    Human demonstrates lexical bias when listening to ambiguous words. This has been argued to be a top-down modulation in human brain. We investigated if this phenomenon is also present in automated speech recognition models, which is trained through a bottom-up, feedforward architecture.
    • Wav2Vec2.0 fine-tuning
    • Speech embedding analysis
    • LLM layer probing

    Journal of Neural Engineering
    Neural signals in the temporal and parietal cortices contain distributed, causally decodable information about speech intent prior to voice onset, offering new avenues to improve speech brain–machine interfaces and extend them to patients with frontal lobe damage.
